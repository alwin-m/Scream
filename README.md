# Scream

An experimental text-based micro-blogging platform built with **PHP** and **MySQL**. Started in **2025**, this project focuses on simple, anonymous expression without profiles, likes, or visuals.

## What It Is

A minimal platform where users post and chat using only text. No identities, no pressure — just words.

## Features

* Anonymous text posts
* Simple, fast feed
* Lightweight PHP backend
* MySQL database
* Easy to deploy

## Contributing

This project is open source. Anyone can contribute ideas, fixes, or improvements.

## Setup

1. Clone the repo.
2. Import the SQL file.
3. Update database config.
4. Run on any PHP-supported server.

## Vision

A space where text matters again — clean, quiet, and honest.
